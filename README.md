# NETFLIX GPT

- Create an app using npx create-react-app
- Configure Tailwind CSS
- App Routes
- Form Validation
- useRef Hook

# Features

- Login/Signup Page
  - Login Form
    - Redirect to Browse Page
- Browse Page ( Post Authentication )
  - Header
  - Top Movie
    - Trailer in background
    - Title & Description of Movie
    - Movies X N ( Horizontally Scrollable)
- Netflix GPT
  - Search Bar
  - Movie Suggestions
